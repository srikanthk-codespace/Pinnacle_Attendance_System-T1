📊 Attendance System

«🚀 A smart web-based attendance management system with authentication, duplicate prevention, and Excel export — built for real-world use.»

---

📌 Project Overview

The Attendance System is a modern web application designed to simplify attendance tracking in educational institutions or organizations.

It allows an admin to securely log in, mark attendance, manage records, and export data — all in a clean and responsive interface.

---

❗ Problem Statement

Traditional attendance systems:

- ❌ Allow duplicate entries
- ❌ Are time-consuming and manual
- ❌ Lack proper data management
- ❌ Do not provide export or analytics

This leads to inefficiency, errors, and poor tracking.

---

💡 Solution

This system solves the above problems by:

- ✅ Secure admin login using Firebase Authentication
- ✅ Preventing duplicate attendance entries
- ✅ Storing data in real-time database
- ✅ Providing date-wise filtering
- ✅ Exporting attendance data to Excel
- ✅ Allowing record deletion
- ✅ Fully responsive UI for mobile & desktop

---

⚙️ Key Features

- 🔐 Admin Authentication
- 📝 Attendance Marking System
- 🚫 Duplicate Entry Prevention
- 📅 Date Filter
- 📊 Excel Export
- 🗑 Delete Records
- 📱 Mobile Responsive Design

---

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Firebase Authentication
- Firebase Realtime Database

---

🧪 System Workflow

1. Admin logs in
2. Enters Name & Roll Number
3. Marks attendance
4. System checks for duplicates
5. Stores data in Firebase
6. Displays records
7. Option to filter or export
